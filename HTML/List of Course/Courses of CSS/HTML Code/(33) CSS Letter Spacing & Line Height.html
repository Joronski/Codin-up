CSS Letter Spacing & Line Height

Description

Line height, also known as leading, is the distance between each line when you have two or more rows of text. Letter spacing, also known as tracking, is the size of the gaps between each letter of each word.

<!DOCTYPE html>

<html>

<head>

<title>Letter Spacing & Line Height</title>

<style>

p.small {
line-height: 0.7;
letter-spacing: -2px;
}

p.big {
line-height: 1.8;
letter-spacing: 3px;
}

</style>

</head>

<body>

<p class="small">

This is a paragraph with a smaller line-height and limited space.<br>
This is a paragraph with a smaller line-height and limited space.

</p>

<p class="big">

This is a paragraph with a bigger line-height and huge space.<br>
This is a paragraph with a bigger line-height and huge space.

</p>

</body>

</html>
