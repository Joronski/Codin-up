CSS Adjacent Selectors

Description

The adjacent sibling selector is used to select an element that is directly after another specific element. Sibling elements must have the same parent element, and "adjacent" means "immediately following".

<!DOCTYPE html>

<html>

<head>

<title>CSS Adjacent Selectors</title>

<style>

div + p {

background-color: yellow;

}

</style>

</head>

<body>

<div>

<p>This is a paragraph</p>

<section>

<p>This is an another paragraph</p>

</section>

<p>This is also an another paragraph</p>

</div>

<p>This is a paragraph not in the div</p>

<p>This is an another paragraph not in the div</p>

</body>

</html>
