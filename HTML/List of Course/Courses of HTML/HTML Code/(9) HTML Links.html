HTML Links

Description

A link is a connection from one web resource to another. A link has two ends, An anchor and direction. The link starts at the “source” anchor and points to the “destination” anchor, which may be any Web resource such as an image, a video clip, a sound bite, a program, an HTML document or an element within an HTML document. You will find many websites or social media platforms ( Like YouTube, Instagram ) which link an image to a URL or a text to a URL etc. In this tutorial we'll show you on how to put links in your webpage.

<!DOCTYPE html>

<html>

<head>

<title>HTML Links</title>

</head>

<body>

<p>Proceed to Facebook: </p>

<a href="https://www.facebook.com/">

<p>Facebook</p>

</a >

</body>

</html>

Code Description

Links are specified in HTML using the “a” tag.

href : The href attribute is used to specify the destination address of the link used. "href" stands for Hypertext reference.

Text link : The text link is the visible part of the link. It is what the viewer clicks on.

Internal & External Links

Description

An internal link is a type of hyperlink whose target or destination is a resource, such as an image or document, on the same website or domain.

While an external link is also a type of hyperlink whose target or destination is another site on the web, such as Facebook, Instagram, Twitter, and lots of other websites out there on the internet.

<!DOCTYPE html>

<html>

<head>

<title>HTML Links</title>

</head>

<body>

<p>Internal Link: </p>

<a href="/Home.html">

<p>Internal Link to Codin'up</p>

</a >

<p>External Link: </p>

<a href="https://www.facebook.com/">

<p>External Link to Facebook</p>

</a >

</body>

</html>

Code Description

The code illustrates the use of Internal & External Links. The internal is a link to the contribute page on Codin'up website. The external is a link to the Facebook website on the World Wide Web.

The Target Attribute in Links

Description

The target attribute is used to specify the location where the linked document is opened.

<!DOCTYPE html>

<html>

<head>

<title>HTML Links</title>

</head>

<body>

<p>If you set the target attribute to "_blank", the link will open in a new browser window or tab.</p>

<a href="https://www.facebook.com/" target="_blank">

<p>Proceed to Facebook</p>

</a >

<p>If you set the target attribute to "_self", the link will open in the same window or tab.</p>

<a href="https://www.facebook.com/" target="_self">

<p>Proceed to Facebook</p>

</a >

<p>If you set the target attribute to "_top", the link will open in the full body of the window.</p>

<a href="https://www.facebook.com/" target="_top">

<p>Proceed to Facebook</p>

</a >

<p>If you set the target attribute to "_parent", the link will open in the parent frame.</p>

<a href="https://www.facebook.com/" target="_parent">

<p>Proceed to Facebook</p>

</a >

</body>

</html>

Code Description

The various options that can be used in the target attribute are listed above in the code.
