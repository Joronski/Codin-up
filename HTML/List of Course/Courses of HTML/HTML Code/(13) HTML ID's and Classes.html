HTML ID's & Classes

Description

In this tutorial, we will know how to identify the specific HTML element by its id using HTML id Attribute, along with understanding its implementation through the examples.

The id attribute is a unique identifier that is used to specify the document. It is used by CSS and JavaScript to perform a certain task for a unique element. In CSS, the id attribute is used using the # symbol followed by id. quotes are not mandatory in tag=” ” in all cases. But writing with quotes is a good practice.

<!DOCTYPE html>

<html>

<head>

<title>HTML ID's & Classes</title>

<style>

#element1 {

color: red;

}

</style>

</head>

<body>

<h1 id="element1">Content 1</h1>

<p>Description of the Content.</p>

</body>

</html>

Code Description

In this example, we simply style the element with id “element1”.

  • The class is an attribute which specifies one or more class names for an HTML element.
  • The class attribute can be used on any HTML element.
  • The class name can be used by CSS and JavaScript to perform certain tasks for elements with the specified class name.

Supported Tags: It supports all HTML elements.

<!DOCTYPE html>

<html>

<head>

<title>HTML ID's & Classes</title>

<style>

.country {

background-color: red;

color: white;

padding: 8px;

}

</style>

</head>

<body>

<h2 class="country">CHINA</h2>

<p>China has the largest population in the world.</p>

<h2 class="country">INDIA</h2>

<p>India has the second largest population in the world.</p>

<h2 class="country">United States</h2>

<p>United States has the third largest population in the world.</p>

</body>

</html>

Code Description

In the above example CSS styles all elements with the class name “country”.
