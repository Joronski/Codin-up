Adding CSS to HTML

Description

In this tutorial, we will know how to identify the specific HTML element by Adding CSS, along with understanding its implementation through the examples.

Cascading Style Sheets (CSS) is used to format the layout of a webpage. With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

Using CSS

CSS can be added to HTML documents in 3 ways:

Inline - by using the style attribute inside HTML elements. An inline CSS is used to apply a unique style to a single HTML element.

<!DOCTYPE html>

<html>

<head>

<title>Adding Inline CSS to HTML</title>

</head>

<body>

<h1 style="color: blue;">A Blue Heading</h1>

<p style="color: red;">A Red Paragraph.</p>

</body>

</html>

Code Description

The following example sets the text color of the <h1> element to blue, and the text color of the <p> element to red.

Internal - by using a <style> element in the <head> section. An internal CSS is used to define a style for a single HTML page.

<!DOCTYPE html>

<html>

<head>

<title>Adding Internal CSS to HTML</title>

<style>

body {

background-color: powderblue;

}

h1 {

color: blue;

}

p {

color: red;

}

</style>

</head>

<body>

<h1>Content 1</h1>

<p>Description of the Content.</p>

</body>

</html>

Code Description

The following example sets the text color of ALL the <h1> elements (on that page) to blue, and the text color of ALL the <p> elements to red. In addition, the page will be displayed with a "powderblue" background color.

External - by using a <link> element to link to an external CSS file. An external style sheet is used to define the style for many HTML pages.

<!DOCTYPE html>

<html>

<head>

<title>Adding External CSS to HTML</title>

<link rel="stylesheet" href="styles.css">

</head>

<body>

<h1>Content 1</h1>

<p>Description of the Content.</p>

</body>

</html>

Code Description

The external style sheet can be written in any text editor. The file must not contain any HTML code, and must be saved with a .css extension.

background-color: powderblue;
