HTML Headings & Text

Description

In this tutorial, we will know HTML Heading, & its implementation through the examples. A HTML heading tag is used to define the headings of a page. Also in this tutorial, we will know the HTML Paragraph, & its basic implementation through the examples.

In our heading tag, there are six levels of headings defined by HTML. 6 heading elements are h1, h2, h3, h4, h5, and h6; with h1 being the highest level and h6 being the least.

  • <h1> is used for main heading. (Biggest in size)
  • <h2> is used for subheadings.
  • If there are further sections under the subheadings then the <h3> elements is used.
  • <h6> for the small heading. (Smallest one)

<!DOCTYPE html>

<html>

<head>

<title>HTML Headings & Text</title>

</head>

<body>

<h1>Heading 1</h1>

<h2>Heading 2</h2>

<h3>Heading 3</h3>

<h4>Heading 4</h4>

<h5>Heading 5</h5>

<h6>Heading 6</h6>

</body>

</html>

Code Description

This example illustrates the HTML heading tags.

Text Formatting

Description

HTML facilitates the ability for formatting text just like we do in MS Word or any text editing software. We will go through a few such options.

<!DOCTYPE html>

<html>

<head>

<title>HTML Headings & Text</title>

</head>

<body>

<b>Bold Text</b>

<strong>Important Text</strong>

<i>Italic Text</i>

<em>Emphasized Text</em>

<mark>Marked Text</mark>

<small>Smaller Text</small>

<del>Deleted Text</del>

<ins>Inserted Text</ins>

<sub>Subscript Text</sub>

<sup>Superscript Text</sup>

<pre>

Text in a pre element
is displayed in a fixed-width
font, and it preserves
both spaces and line breaks

</pre>

</body>

</html>

Code Description
  • The HTML <b> element defines bold text, without any extra importance.
  • The HTML <strong> element defines text with strong importance.
  • The <i> tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.
  • The HTML <em> element defines emphasized text.
  • The HTML <small> element defines smaller text.
  • The HTML <mark> element defines text that should be marked or highlighted
  • The HTML <del> element defines text that has been deleted from a document.
  • The HTML <ins> element defines a text that has been inserted into a document.
  • The HTML <sub> element defines subscript text.
  • The HTML <sup> element defines superscript text.
  • The HTML <pre> tag defines preformatted text.
